# Studio Intelligence Platform - Miro Board Kit

Files in this folder:

- studio-intelligence-platform-miro-board.svg: upload/drag directly into Miro as the full board visual.
- studio-intelligence-platform-board.drawio: opens in diagrams.net/draw.io; useful as a fallback export route.
- database-design.csv: database table export for spreadsheet/table import.
- miro-sticky-notes.csv: structured card/sticky-note backup.

## Import Notes

Recommended path: drag the SVG into Miro. It lands as one complete board image.

If you need a fallback, open the .drawio file in diagrams.net and export as SVG, PNG, or PDF, then upload that to Miro. Use the CSV files if you want to rebuild sections as editable cards or tables.
